Viewport

The viewport is the visible area of a web page on a device's screen. It varies depending on the device and is controlled in responsive design using the meta viewport tag.

Flexbox

Flexbox is a one-dimensional layout system that arranges items in rows or columns and distributes space efficiently within a container.

Grid

A grid layout is a two-dimensional layout system that allows designers to organize content into rows and columns simultaneously.
